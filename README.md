# S S Enterprises - Manpower Consultancy Website

A fully complete, professional, and responsive corporate web platform built using **Bootstrap 5**, native modern **CSS3**, and **Vanilla Javascript**.

## Core Visual Features Built
- **Sticky Navigation Menu & Top Bar**: Quick structural details extraction matching Jagat Trade Center, Patna address mapping perfectly.
- **Hero Intro Panel**: Designed meticulously around specialized blue-and-white enterprise corporate metrics.
- **Fully Built Sub-sections**: Contains functional divisions for About Us, Services Portfolio, Industry Coverage matrix, Call-To-Actions, and an embedded infrastructure mockup framework.
- **Dynamic Action Overlays**: Features a permanent floating WhatsApp integration wrapper, direct phone call triggers, and interactive scroll back-to-top setups.
- **Google Maps Integrations**: Deep link embedded system maps routing straight to Frazer Road, Patna landmarks securely.
